HTML是什么

    HTML:其实就是HyperText Mark-up Language,中文就是超文本标记语言

     超文本释义：不仅能显示文本，还能够支持链接、图片、视频等内容
           但是内容需要在项目同一文件夹内才可以调用
     标记释义:标记也成为标签，也就是符号
     标记语言释义：通过标签描述文本结构及其他信息的符号系统
           使用标记主要是方便浏览器对所写的内容在网页显示
     结论一：网页的本质就是文本
     结论二：浏览器会默认解析HTML或HTM后缀名的文件
     结论三：浏览器根据“标签”，将网页内容进行不同形式的显示
     总结：学习HTML,就是学习标签及其效果

不同浏览器的由来与区别

  • 浏览器的由来

    浏览器源自WebKit开源的一个专注于网页内容展示所开发出的一流网页渲染
    引擎而后谷歌以webkit为核心，并发起了一个Chromium的浏览器开源项目再
    接着谷歌公司优化了webkit并推出了自己的Blink引擎，也就是Chrome浏览器

  • 浏览器的区别

    现在市面上的所有浏览器基本没有区别，国内的浏览器都是用老外的不同类型内
    核形成的双核浏览器，例如：Trideng+Webkik或者Blink

  • 浏览器的解析标准由来

    为了不同浏览器渲染相同代码的页面显示一样万维网发明人
    蒂姆·博纳斯·里 在1994年10月成立的W3C（万维网联盟组织）
    制定了HTML CSS XML 等网络标准以此希望实现相同的网页
    代码在各个浏览器上显示都一样。

HTML标签的规定：

  • Html标签是由尖括号包围的，例如：

     <html></html>
     <img/>：英文尖括号

  • 标签的书写方式，分为双标签和单标签，例如：

     双标签：<body></body>
     单标签：<img/>

  • 标签可以嵌套，但不能交叉嵌套，例如：

     <html>
      <head>
      </head>
     </html>
    以上为正确表达式
     <html>
      <head>
     </html>
      </head>
    以上为错误表达式

  • 标签中可以包含属性，例如：

     <img src="文件名.xxx">
     src 就是属性名
     文件名.xxx 就是属性值

  • 标签不区分大小写，一般是小写，例如：

     常见的：<html></html>
     不常见：<HTML></HTML>

网页的基本结构：

  • 示例

    <!DOCTYPE html>
    <html>
     <head>
      <meta charset="UTF-8">
      <title>网页标题</title>
     </head>
     <body>
        网页内容
     </body>
    </html>
